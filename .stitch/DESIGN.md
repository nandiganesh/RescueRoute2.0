# RescueRoute Design System

## Global Style & Vibe
A modern, minimal, clean mobile UI inspired by apps like Uber Eats. Lightweight, fast, usable, and clear. No dark mode. Smooth, simple layout without clutter. Fast readability.

## Color Palette
* **Primary:** Soft Green (#2ecc71) - used for primary actions like Accept, Complete.
* **Secondary:** Light Green (#a8e6cf) - used for subtle highlights or secondary elements.
* **Background:** Off-white (#f9fafb) - base background for all screens.
* **Cards & Containers:** Pure White (#ffffff) - for main content blocks.
* **Text:** Dark Gray (#333333) - for high contrast readability.

## Typography
* Modern sans-serif (e.g., Inter, Roboto).
* Clean, legible at small sizes.

## UI Components
* **Cards:** Rounded corners (approx 12px-16px radius) with soft, subtle drop shadows.
* **Buttons:** Large tap targets (min 48px height), rounded corners, primary color for main action, secondary subtle styling for decline.
* **Bottom Sheets:** Floating or anchored panels at the bottom with a drag handle, rounded top corners, shadow on top.
* **Badges:** Small rounded pills for status (Available, Delivering, Completed) with appropriate text/background colors.

## Spacing & Layout
* Mobile screen proportions (iPhone).
* Plenty of white space.
* Clean spacing (multiples of 4/8px).
* Clear visual hierarchy.
